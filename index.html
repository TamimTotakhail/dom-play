To fulfill the requirements of this assignment, we created a single page application using the DOM and JavaScript.

When we click on a specific actor's name or any associated element, all the lines belonging to that actor in a play will be highlighted. The highlighting can be achieved by applying a bold style or any other visual indicator.

The function(s) we created should be able to handle any number of actors and any additional lines that may be added to the page dynamically. This ensures that the highlighting and un-highlighting will work correctly regardless of the number of actors or lines.

The page will perform the required actions without reloading the page, using JavaScript and manipulating the DOM directly.

We made necessary modifications to the HTML structure to enable the JavaScript scripts to function properly.
